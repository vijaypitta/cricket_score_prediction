# cricket_score_prediction
Cricket Live Score Prediction
Overview

This project predicts the live score of a cricket match based on historical data and current match conditions. The goal is to estimate the score in real-time by analyzing various factors like batting performance, bowling stats, weather, and match phase.

Features

Real-time score prediction based on match data.

Uses machine learning models trained on historical match data.

Displays estimated score as the match progresses.

Installation

Clone the repository:

git clone https://github.com/yourusername/cricket-live-score-prediction.git
cd cricket-live-score-prediction


Create a virtual environment and activate it:

python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate


Install the dependencies:

pip install -r requirements.txt


Run the app:

python app.py

Requirements

Python 3.6+

Required libraries listed in requirements.txt

Usage

Run the app, and it will predict the live score based on the current match conditions.
